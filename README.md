LinkedList exercise for the Odin Project.

Creates a singly linked list implementing the following methods:
- `append(value)` adds a new node containing `value` to the end of the list.
- `prepend(value)` adds a new node containing `value` to the start of the list.
- `size` property holds the total number of nodes in the list.
- `head` property holds the first node in the list.
- `tail` property holds the last node in the list.
- `at(index)` returns the node at the given `index`.
- `pop` removes and returns the last element in the list.
- `contains(value)` returns true if `value` is in the list, otherwise false.
- `find(value)` returns the index of the node containing `value`, or null if not found.
- `toString` returns a string representation of the LinkedList.
- `insertAt(value, index)` inserts a new node containing `value` at the provided `index`. Out of range values for `index` are constrained to the LinkedList's present values.
- `removeAt(index)` removes the node at `index`. Out of range values for `index` are constrained to the LinkedList's present values.
